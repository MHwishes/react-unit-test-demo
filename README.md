# Redux Todos Example

### how to run the demo 
 `git clone git@github.com:MHwishes/react-unit-test-demo.git`
 `cd react-unit-test-demo`
 `npm i`
 `npm start`
Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### how to run test
``
npm run test
``
