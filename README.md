# Redux Todos Example

### set up the project
 `git clone git@github.com:MHwishes/react-unit-test-demo.git`

 `cd react-unit-test-demo`

 `npm i`

 `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run test`

Run all the test
